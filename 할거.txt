api 서버와 클라이언트 간 서비스 상호작용
query 데이터 조회 (읽기 전용)
mutaion 데이터 변경 요청(추가,수정,삭제)
hook 함수형 컴포넌트(api등)의 호출 결과를 상태로 관리.

setSearchParams url파라미터의 객체, set을 하면 url변경


@ModelAttribute
url파라미터를 자동으로 맵핑, get, post요청
@RequestBody
post, put 요청
@RequestParam
url파라미터나 폼데이터를 처리. get요청

style={{}}에서 중괄호{{를 두번 쓰는 이유는
자바스크립트 코드가 JSX 문법 내에서 실행될 때,
style={}내부의 {}를  React가 객체 형태로 CSS 스타일을 인식하기 위해서이다.
